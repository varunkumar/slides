Node.js

  • Platform built on Chrome's JavaScript runtime (V8) for easily building fast, scalable network applications.
  • Event driven "Server Side" JavaScript
  • Non-blocking I/O. Async model.
    • Event loop
    • libev, libeio
  • Authored by Ryan Dahl. Sponsored by Joyent
  • Alternatives: Twisted for Python, Event Machine for Ruby

Node.js - Concurrency

  • I/O bound vs CPU bound
  • Everything is asynchronous except your code
  • Non-blocking I/O. Async model.
    • Event loop
    • libev, libeio
  • Single stack environment
  • Multi-processor concurrency?
    • child_process.fork()
    • For load balancing, use cluster module
